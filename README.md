# Snake Game (C++ & raylib)

A classic Snake Game developed in C++ using the raylib graphics library.

## Features

- Snake movement using arrow keys
- Random food generation
- Score tracking
- Collision detection
- Sound effects
- Game over and restart

## Technologies

- C++
- raylib
- Makefile

## Controls

- ↑ Move Up
- ↓ Move Down
- ← Move Left
- → Move Right

## Installation

```bash
git clone https://github.com/mokshithakiran/SnakeGame.git
cd SnakeGame
make
./SnakeGame
```
